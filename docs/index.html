About

BP:Essentials is made By UserR00T And DeathByKorea to make it easier for server owners to moderate their servers.             
This plugin is in active development, If you encounter any bugs, please submit them on the GitHub!

Commands

You can change all commands, aliases, and the commandchar (/) in the settings file. (Essentials/essentials_settings.txt)
Tip: You can press the ▶ for more info about that command/feature.

Default commands:

Command Alias Description
/reload /rl Reloads all files and commands
/clearchat [everyone/self] /cc Clears the chat for yourself or for everyone
/broadcast [message] /say Broadcasts a message to the whole server.
/mute [Username/ID] /unmute Mutes the player server-wide
/fakejoin [username] /fakeleave Prints a join/leave message with the given username
/checkalts [ip/ign] [Username/IP] /ca Check the IP of a player or accounts associated with an IP
/godmode /god Makes you invincible.
/discord n/a Prints the servers discord to the chat!
/heal [Username/ID] /hp Changes the HP of [Username/ID] to 100.
/feed [Username/ID] /maxstats Maxes all stats of [Username/ID].
/info [Username/ID] /stats Gives information about the player eg, HP, Job, IsAdmin, IP, etc.
/clearwanted [Username/ID] cw This command resets the players wanted level.
/give [ID] [AMOUNT] /spawn Gives an item to the player.
/latestvoteresults /lvr Gets a list of players who voted yes on the latest vote.
/atm n/a Opens your bank account. Can be disabled. (Just like all other commands.)
/pay [Username/ID] [Amount] /send Pays a user [amount] of money. The money must be in your inventory.
/money [Username/ID] [amount] n/a Spawns in [amount] of money.
/slap [Username/ID] /slp Damages a player a little bit (between 4 and 15) and moves him around.
/strip [Username/ID] /rc Removes all clothes from an player.
/launch [Username/ID] takeoff Launches an player into the air.
/setjob [JobID/JobName] [Username/ID] /sj
Sets the job of a player to:
0: Citizen
1: Criminal
2: Prisoner
3: Police
4: Paramedic
5: Firefighter
6: GangsterRed
7: GangsterGreen
8: GangsterBlue
9: Mayor
10: DeliveryDriver
11: TaxiDriver
/tphere [Username/ID] /tph Teleports a player to you.
/logs n/a Gets latest chat logs. NOTE: This command is very unstable. Limted to last 100 messages.
/save n/a Saves all current connected players progress. NOTE: It does not save while the player is in the apartment.
/report [Username/ID] rport
Opens an UI where the player can report someone, after they did that it gets send to all online staff.
/search [Username/ID] srch Searches [Username/ID].
/kill [Username/ID] n/a Kills [Username/ID].
/ban [Username/ID] n/a Bans [Username/ID].
/kick [Username/ID] n/a Kicks [Username/ID].
/free [Username/ID] n/a Frees [Username/ID].
/arrest [Username/ID] n/a Arrests [Username/ID].
/restrain [Username/ID] n/a Restrains [Username/ID].
/jail [Username/ID] [TimeInSeconds] placeinjail Jails [Username/ID] for [TimeInSeconds].
/knockout [Username/ID] ko Knocks out [Username/ID].
/afk n/a Marks you as AFK. When someone types your name they will get a message saying that you're afk.
/createkit [DelayInSeconds] [Price] [KitName] /ck Creates a new kit if it doesn't already exist.
/kits /listkits Lists all the kits you have access to and aren't disabled.
/kit [Name] /getkit Gives you that kit as long as you have permission to it and it isn't disabled.
/vehicle [id] /v Spawns in vehicle with that id.
/tplocation [location] /tpl Teleports you to a specific location.
/tpall n/a Teleports everyone to your position.
/tpapartment [10k, 5k, 1.2k n/a Teleports you to a specific apartment. (you can also use /tplocation [10k, 5k, 1.2k])
/spy /spycommands Will log all commands users enter to your chat.
/promote [Username/ID] /rankup Promotes a user to the their next rank (if possible).
/clear [Username/ID] /clearinv Clears a user inventory completely.
/disablestaffchat /dsc Disable recieving staff messages.
/staffchat /sc Toggles staffchat.
/t /tchat Toggles chat.
/createwarp [DelayInSeconds] [Price] [WarpName] /ck Creates a new warp if it doesn't already exist.
/warps /allwarps Lists all the warps you have access to and aren't disabled.
/warp [Name] /gotowarp Teleports you to that warp as long as you have permission to it and it isn't disabled.
/tpa [Username/ID] /tprequest Sends a teleport request to a user. If that user accepts the request you'll be teleported to them.
/tpaccept /tpaa Accepts last teleport request.
/tpadeny /tpad Denies last teleport request.
/pm [ID] [Message] /pmuser Sends a direct message to a user.
/r [Message] /ruser Responds to the last direct message.
/disconnect [Username/ID] /dconnect Disconnects someone from the server but does not temp ban them.
/home /h Teleports you to your apartment.
/wipeserver [Password] /unleashallchaos Deletes all user data. (Password can be set in config.)

Features

Tip: You can press the ▶ for more info about that command/feature.

Feature Description
LanguageBlock and ChatBlock Block rasist or messages in other languages!
Broadcast messages to the server Get players attention and distinguish yourself as a moderator!
Clear the chat and remove spam Clear the chat locally or globally to remove spam or words not in the ChatBlock!
Log IPs of players Logs the IPs of players so you can ban by IP!
Announcements Announce messages periodicaly from a predefined list!
Username Banning Ban players by username!
Admin chat color
Admins get a cool color in chat to distinguish yourself from players!
UI Menus (FunctionUI)
You can press F1 to F11 to open a menu with alot of handy things!
Open for screenshots
Block votekicking You can completely disable votekicking though the config.
Confirm to sell your apartment Before you can sell your apartment you have to type /confirm.
Custom Commands Add custom commands though custommcommands.txt! When the command gets executed, it will send the response back to the player.
JSON Config A config where you can change alot of stuff like who can execute a command, message colors, and loads more!
Custom Groups Custom Groups will allow groups to have a custom color in chat, and permission to specific commands! For more info, check Essentials/CustomGroups.txt

Installation



Installing the plugin

Assuming you have a server already;
Download the plugin from this link: Download
Open the zip.
Move "Assembly-CSharp.dll", "BP-Essentials.dll", "Newtonsoft.Json.dll", and "System.Runtime.Serialization.dll" to "YourServerFolder\BrokeProtocol_Data\Managed".
When asked, press Overwrite.
Done! Run your server once to get all required files.
Start configurating the settings file in "YourServerFolder\Essentials\essentials_settings.txt".
Optional: Download BP-ConsoleMonitor from here
You can use BP-ConsoleMonitor to monitor your chat, commands, login and logouts, and alot more!

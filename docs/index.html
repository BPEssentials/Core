About

BP:Essentials is made By UserR00T And DeathByKorea to make it easier for server owners to moderate their servers.             
This plugin is in active development, If you encounter any bugs, please submit them on the GitHub!

Commands

You can change command names in the settings.txt file.

Default commands:

Command Alias Description
/reload /rl Reloads all files and commands
/clearchat /cc Clears the chat for yourself or for everyone
/broadcast /say
Broadcast to the whole server
/godmode /god Makes you invulnerable for damage


More commands and features coming later!

Features

Feature Description
LanguageBlock and ChatBlock Block those rasist or messages in other languages!
Broadcast messages to the server Get players attention and distinguish yourself as a moderator!
Clear the chat and remove spam Clear the chat locally or globally to remove spam or words not in the ChatBlock!

Installation



Installing UniversalUnityHooks

Assuming you have a server already;
Download UniversalUnityHooks from this link: https://github.com/DeathByKorea/UniversalUnityHooks/releases
Extract the zip somewhere
Place the HookAttribute.dll in $Gamedir/Game_Data/Managed
Place HooksInjector.exe along with the Mono.Cecil dll's (Mono.Cecil, Mono.Cecil.Inject, Mono.Cecil.Mdb, Mono.Cecil.Pdb, Mono.Cecil.Rocks) in $Gamedir

Follow intruction down here for the plugin installation



Installing the plugin

Assuming you have a server already and UniversalUnityHooks installed;
Download the plugin from this link: Download
put the Essentials.cs file in the scripts folder
Done! Run it once and start configuring the Essentials/settings.txt file
Reload with the /reload command
